# Job Hub Platform

## Project Overview
Job Hub is a full-stack MERN application designed to connect job seekers with employers. It allows users to register, browse and apply for jobs, and receive notifications for application status. Employers can post jobs and manage applicants.

## Features
- User authentication (register/login)
- Job posting (for employers)
- Job browsing and applying (for job seekers)
- Notifications for job applications
- Responsive React frontend
- Secure backend with Node.js, Express.js, MongoDB
- JWT-based authentication

## Tech Stack
- **Frontend:** React.js, Axios, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Atlas or local)
- **Authentication:** JWT
- **Deployment:** Frontend (Vercel/Netlify), Backend (Render)
